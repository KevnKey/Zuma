Project Zuma
---------------
Project Zuma is a recreation of the original game Zuma where the player shoots colored balls
at an incoming line of balls in order to eliminate all of them.

To run the game: Right click on the python file 'Project Zuma' and select open with Visual Studio Code.
Then press ctrl + b

Testing Shortcuts:
'p' - Pauses all balls on the path but not the moving balls
'Space' - Pauses the game
